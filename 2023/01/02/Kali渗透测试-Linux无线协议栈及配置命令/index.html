802.11协议栈

  • 协议栈
    • 内核接口（kernel）
    • 用户空间命令及相应的调用程序（userspace）

无线网卡配置

查看无线网卡

  • ifconfig
  • iwconfig
  • iw list

其中Mode模式为Managed，只有Monitor模式才能抓包

iw list命令：列出网卡支持的加密方法，接口模式，信道，功率以及命令等

其中支持的接口模式中必须要有monitor和AP两种接口，monitor作为抓包接口，AP作为eval AP用来作为流氓AP

查看信道频率

  • iwlist wlan0 frequency
  • iw list

扫描AP

  • iw dev wlan0 scan
  • iw dev wlan0 scan | grep SSID
  • iw dev wlan0 scan | egrep “DS Parameter set|SSID”
  • iwlist wlan0 scanning | egrep “ESSID|Channel”
  • iw dev wlan0 scan

  • iw dev wlan0 scan | egrep “DS Parameter set|SSID”

  • 每一个BSS就是一个无线AP，后面会跟着相应的mac地址，以及此AP的各种详细信息；
  • beacon interval每100毫秒发送一个beacon，1秒发送10个beacon；
  • signal表示信号强度，负值表示接收信号强度
  • channel 6表示使用信道6
  • Authentication suites: PSK表示身份验证方式是PSK（预设的共享密钥）；知道共享密钥就可以连接AP

添加删除侦听端口

  • iw dev wlan0 interface add wlan0mon type monitor（注意命名限制：wlan+数字+mon）
  • tcpdump -S 0 -i wlan0mon -P(建议用wireshark抓包查看)
  • iw dev wlan0mon interface del
添加无线接口

  • iw dev wlan0 interface add wlan0mon type monitor

    add后面是新添加的接口名称wlan0mon；类型type是monitor

    此时已经成功添加一个新的无线网卡，类型为Monitor

    使用ifconfig wlan0mon up是使其活跃起来

    注意：如果” iw dev wlan0 interface add wlan0mon type monitor “命令执行完 wlan0mon 是 Managed 模式或” ifconfig wlan0mon up”报错（参见下图），则可能是 network-manager 对 iw 命令相冲突，可通过先执行” service network-manager stop”命令，再进行如上操作解决。

使用wireshark抓包

此时就能抓到无线数据包了

删除无线接口

  • iw dev wlan0mon interface del

    此时已经删除了wlan0mon无线网卡
